AOM Trading

Category

WordPress, UI/UX, Web Design

Posted

December 2020

AOM Trading is a site for people to learn how to trade through class subscriptions. I worked with the founder, Alex Minkovich, to design and implement features for a clean user experience.

I designed class pages, worked on user flows, and integrate my designs into WordPress. This is one of my first project, and it showcases my expertise in WordPress. I have learned a lot since this project.

View:View site

Empathize

In order to understand the wants and needs of the company, Alex and I went over the users of the companies, so I could understand how users would interact with the site. The goal of this website is to provide access to educational trading recources. The challenge is taking complicated concepts and making it more easily approachable for users to begin their journey to learn how to trade. It's improtant to build trust, confidence, and ease of use for the clients.

Competitive Analysis

Alex and I walked through competitor sites to get a sense of how we could organize the information.

Learning WordPress

I used WordPress plugins to integrate a subscription system for clients to sign up. used CSS to manipulate the user interface deign as well and make the site clean.

Video Editing

I used Adobe Premiere and iMovie to edit class videos for AOMTrading. I would then post these videos onto AOMtrading.com. I created the intro animation and worked through the videos to cut out any gaps to shorten video length and increase the amount of time teaching content in the videos.

Results

After collecting user insights, we learned that customers want supplemental one-on-one help so we worked on integrating tutoring into the AOM Trading website. It is not a class substitution but rather a clarification of classroom materials.
